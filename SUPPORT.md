# Support

## Getting Help

Use project documentation first:

- `README.md`
- `docs/USER_GUIDE.md`
- `docs/DEVELOPER_GUIDE.md`
- `docs/POLICY_MODEL.md`

For bugs and feature requests, open an issue with:

- Reproduction steps
- Tool version
- Policy and rules used
- Expected and actual behavior

## Support Scope

- Usage questions
- Configuration and policy tuning
- Integration guidance for CI/CD
- Bug triage for supported versions

## Out of Scope

- Proprietary environment debugging without reproducible steps
- Custom one-off rule authoring services
- Urgent SLA-backed support unless explicitly contracted
